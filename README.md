# FormBuilder 1.0

Dependancies :

  * jQuery
  * jQuery UI
  * Undersocre
  * Backbone
  * Less
  * Bootstrap 2.3.2
  * Metro-bootstrap

### How to import the form builder

The simpliest way is :

  1. Add this HTML element to your body : 
    
    <section id="formBuilder" class="container-fluid"></section>

  2. Add the following JS code : 
    
    
    
    $(document).ready( function() {
        formBuilder.init();
    });
